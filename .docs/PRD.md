# 마이링크 (MyLink) - 제품 요구사항 정의서 (PRD)

## 1. 프로젝트 개요
- **프로젝트명:** 마이링크 (MyLink)
- **목적:** 여러 소셜 미디어, 웹사이트, 포트폴리오 등의 링크를 하나의 통합된 페이지로 관리하고 공유할 수 있는 링크트리(Linktree) 클론 서비스 구축
- **주요 기술 스택:** Next.js (웹 프레임워크) + Firebase (인증, 데이터베이스, 스토리지 등 기능 구현)
- **서비스 정책:** 모든 기능 전면 무료 제공 (별도 결제 티어 없음)
- **목표 일정:** 별도의 엄격한 데드라인 없이 기능 완성도와 품질을 우선시하여 진행
- **대상 사용자:** 
  - 다양한 채널(인스타그램, 틱톡, 유튜브 등)을 운영하는 크리에이터 및 인플루언서
  - 포트폴리오 및 이력서를 하나의 링크로 공유하고 싶은 프리랜서 및 구직자
  - 다수의 링크 공유가 필요한 소상공인 및 비즈니스 계정 운영자

## 2. 핵심 기능 목록

### 필수(MVP) 기능
1. [회원가입 및 로그인] - 이메일 기반 기본 인증 시스템
2. [프로필 관리] - 프로필 이미지, 닉네임, 한줄 소개 설정
3. [링크 관리] - 링크 추가, 수정, 삭제, 순서 변경(Drag & Drop)
4. [퍼블릭 페이지] - 고유 URL(`/username`)을 통한 사용자 페이지 제공

### 선택(고도화) 기능
1. [테마 및 스타일 커스텀] - 배경색, 버튼 스타일, 폰트 변경
2. [통계 수치 제공] - 페이지 방문자 수 및 링크별 클릭 수 확인
3. [소셜 로그인] - 구글, 카카오 등 간편 로그인 지원
4. [예약 및 스케줄링 링크] - 특정 시간대에만 링크 노출

## 3. 각 기능의 상세 설명

### 3.1 필수(MVP) 기능 상세
- **회원가입 및 로그인**
  - 사용자는 이메일과 비밀번호를 통해 계정을 생성할 수 있습니다.
  - **이메일 인증 필수:** 가입 시 해당 이메일의 유효성을 확인하기 위한 인증 절차(인증 링크 또는 코드)를 필수로 거쳐야 합니다.
  - 가입 시 고유한 `username`(페이지 URL에 사용될 ID)을 설정해야 하며, 중복 검사가 필수적입니다.
- **프로필 관리**
  - 마이페이지 내에서 본인의 개인 프로필 이미지(아바타)를 업로드하거나 삭제할 수 있습니다.
  - 페이지 상단에 표시될 닉네임과 짧은 소개글(Bio)을 작성하고 수정할 수 있습니다.
- **링크 관리**
  - **추가:** 직관적인 UI를 통해 표시할 타이틀(Title)과 실제 URL을 입력하여 새 링크 항목을 생성합니다.
  - **수정/비활성화/삭제:** 등록된 링크의 내용을 언제든지 수정하거나 임시로 숨김 처리(Toggle) 및 완전 삭제할 수 있습니다.
  - **순서 변경:** 드래그 앤 드롭(Drag & Drop) 인터페이스를 통해 사용자 페이지에 보여질 링크 노출 순서를 변경합니다.
- **퍼블릭 페이지 제공**
  - `mylink.com/username` 형식의 접속 가능한 퍼블릭 뷰(공개 페이지)를 동적으로 생성하여 방문자에게 제공합니다.
  - 기본적으로 모바일 해상도에 최적화된 반응형 UI로 구성됩니다.

### 3.2 선택(고도화) 기능 상세
- **테마 및 스타일 커스텀**
  - 사전 정의된 3~5가지의 테마(색상, UI 구성) 프리셋을 제공합니다.
  - 배경(단색, 그라데이션), 버튼 모양(둥근 모서리, 각진 모서리), 폰트 스타일 등을 개별 설정할 수 있습니다.
- **통계 수치 제공 (Analytics)**
  - 관리자 대시보드에서 본인 페이지의 총/일일 조회수(Page Views)를 시각적인 차트로 확인할 수 있습니다.
  - 각 링크별 클릭 수(Click-throughs) 데이터를 제공하여 어떤 링크의 전환율이 높은지 분석하도록 돕습니다.
- **소셜 로그인**
  - OAuth 2.0 기반으로 구글, 카카오, 네이버 계정을 연동하여 원클릭 회원가입/로그인 경험을 제공합니다.
- **예약 및 스케줄링 링크**
  - 특정 날짜와 시간에만 링크가 활성화되거나 특정 일자에 자동으로 사라지도록 설정합니다(예: 프로모션/이벤트 링크).
