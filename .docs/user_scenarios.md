# 마이링크 (MyLink) - 사용자 시나리오

## 1. 방문자: 마이링크 페이지 방문 과정
* **접속:** 방문자는 소유자의 여러 소셜 정보와 콘텐츠를 한곳에서 파악하기 위해 제공된 소유자의 마이링크 URL(`mylink.com/username`)로 접속한다.
* **정보 식별:** 방문자는 이 페이지가 자신이 찾던 사람의 것이 맞는지 검증하기 위해 상단의 프로필 이미지, 닉네임, 그리고 한 줄 소개를 확인한다.
* **콘텐츠 이동:** 방문자는 특정 목적으로 제공된 외부 서비스나 포트폴리오로 넘어가기 위해 화면 내에 나열된 링크(버튼) 중 원하는 항목을 클릭한다.
* **서비스 가입 유입:** 방문자는 깔끔하게 정리된 페이지를 경험한 뒤 자신만의 페이지를 개설하기 위해 화면 맨 아래에 위치한 '마이링크 로고(만들기 버튼)'를 클릭하여 가입 페이지로 이동한다.

## 2. 소유자: 링크 관리 (추가/수정/숨김/삭제/정렬) 과정
* **추가 시작:** 소유자는 자신의 새로운 채널이나 결과물을 팬/방문자에게 즉시 공유하기 위해 관리자 대시보드에서 '링크 추가' 버튼을 클릭한다.
* **정보 기입:** 소유자는 방문자가 클릭 시 이동할 명확한 목적지와 의도를 안내하기 위해 새 링크의 '타이틀(제목)'과 '도착 URL 주소'를 입력한다.
* **수정:** 소유자는 기존에 등록된 링크의 제목이나 URL 링크 주소를 최신 상태로 갱신하기 위해 해당 링크의 '수정' 영역을 탭하여 내용을 바꾼다.
* **숨김(비활성화):** 소유자는 나중에 다시 사용할 링크 정보를 지우지 않고 일시적으로 방문자에게만 보이지 않게 감추기 위해 해당 링크의 활성화 상태(Toggle)를 끈다(Off).
* **삭제:** 소유자는 더 이상 방치되거나 유효하지 않은 링크가 방문자에게 노출되는 것을 막기 위해 해당 항목의 '삭제' 버튼을 눌러 리스트에서 완전히 제거한다.
* **순서 변경:** 소유자는 방문자가 가장 중요하게 확인했으면 하는 핵심 콘텐츠를 페이지 최상단에 보여주기 위해 마우스 드래그 앤 드롭으로 링크 노출 순서를 직관적으로 바꾼다.
